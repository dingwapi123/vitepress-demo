# 什么是文档中心？

文档中心是一个专业的技术文档管理平台，旨在帮助团队更好地管理和共享知识。

## 核心特性

- 简单易用的文档编辑
- 实时协作功能
- 版本控制
- 强大的搜索功能

## 适用场景

- 技术文档管理
- API 文档
- 产品使用手册
- 开发规范
